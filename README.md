Name of the Project DigiTools – Digital Tools Marketplace

A Little Description: DigiTools is a modern, single-page marketplace web app where users can browse and purchase premium digital products such as AI writing assistants, design template packs, automation toolkits, resume builders, and social media content kits. It features a clean, responsive UI with a product catalog, a shopping cart, pricing plans, and a smooth "how it works" flow — giving users a complete e-commerce style experience for digital tools.

Technology Used:
React 19 – component-based UI
Vite – fast build tool and dev server
Tailwind CSS + daisyUI – styling and theming
React Toastify – toast notifications
localStorage – to persist cart data

3 Key Features:
Product Marketplace with Tabs – Browse a curated collection of digital products and switch between a "Products" view and a "Cart" view using a clean toggle interface.
Persistent Shopping Cart – Add or remove items from the cart with a live item counter, and the cart automatically saves to localStorage so it stays intact even after a page refresh.
Fully Responsive, Modern UI – A polished landing experience with a hero section, live stats, step-by-step guide, and pricing plans, all designed with smooth animations and mobile-friendly layouts.
